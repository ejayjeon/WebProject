# Jemstagram
# 목차
## 1. 개발툴 (Development Tool)
* IDE : Visual-studio
* Language : JavaScript (Node.js)
* Front-end tool : Vue3 / vuex / vue-router4
* Database : Firebase (Firestore / Storage / Functions)
* UI Framework : Quasar 
* Hosting tool : Github
## 2. 컬렉션 모델 (Collection Model)
### 2.1 Post
* name : String
* profileImage : String
* postImage : String
* postContent : String
* PostDate : String
* Likes : Number
* filter : String

### 2.2 User
* Email : String
* name : String
* profileImage : String
* userMessage : String
* private : false
